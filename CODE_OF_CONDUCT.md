# Contributor Covenant Code of Conduct (Short)

All contributors are expected to adhere to this project's code of conduct.

Be respectful. Be kind. Harassment and abusive behavior are not tolerated.

If you experience or witness unacceptable behavior, please open an issue or contact the repository owner.

This is a short form of the Contributor Covenant. For the full text, see: https://www.contributor-covenant.org
